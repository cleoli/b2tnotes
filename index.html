Introduction to HTML

Sublime is a software that we can use to write our codes. You can right click on your coding area to select the option to open your file in browser.

An HTML element usually consists of an opening tag and a closing tag, e.g. <html> </html>. The backbone elements of an HTML file are the following:

<!DOCTYPE html>
<html>

<head>
</head>

<body>
</body>

</html>

The body element of an HTML document holds all the content of web page for display. The head element of an HTML document holds the meta data, e.g. CSS files, etc.

A few other HTML elements:

<title> </title> will go inside the head element, will be displayed on the tab of a brower page
<h1> </h1> is an heading element and will go inside the body element
<h6> </h6> is the smallest heading element
<img> is a self contained element and will be used for displaying an image
<p> </p> is a paragraph element and will go inside the body element

Tag Format:

Tag name, attribute
Common HTML attributes: alt, disabled, href, id, src, style, title
